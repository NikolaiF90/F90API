## What is F90 API?

F90 API is a shared scripting foundation for AI Dungeon. It gives every script a common set of tools — character management, text utilities, notifications, story card access — so developers stop reimplementing the same logic and world makers stop wrestling with compatibility.

Scripts built on F90 API speak the same language. Drop two compatible scripts into the same scenario, register them, and they run together without conflict. No merging, no copy-pasting, no guessing.

Every method on F90 API was proven necessary by real scripts before being added. Nothing speculative, nothing designed upfront. Battle-tested before it ships.
